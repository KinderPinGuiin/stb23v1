STB23 help

[GET] /

Display the project information.

[GET] /help

Display the API help.

[GET] /stb23/resume/xml

Display a summary of all the STB inside the database (XML format).

[GET] /stb23/resume

Display a summary of all the STB inside the database (HTML format).

[GET] /stb23/xml/{id}

Display the content of the STB identified by {id} (XML format).

[GET] /stb23/html/{id}

Display the content of the STB identified by {id} (HTML format).

[POST] /stb23/insert

Insert the given STB inside the database.

[DELETE] /stb23/delete/{id}

Delete the STB identified by {id} from the database.
